1️⃣  CREATE SUPABASE ACCOUNT
    → Go to: https://supabase.com
    → Click "Start your project"
    → Sign up with GitHub or Email

2️⃣  CREATE NEW PROJECT
    → Click "New Project"
    → Name: codeconvert-pro
    → Choose a strong database password
    → Select region (nearest to you)
    → Wait ~2 minutes for setup

3️⃣  GET YOUR CREDENTIALS
    → Go to Settings → API
    → Copy "Project URL" 
    → Copy "anon public" key

4️⃣  CONFIGURE YOUR APP
    → Open: supabase-config.js
    → Replace 'YOUR_SUPABASE_PROJECT_URL' with your URL
    → Replace 'YOUR_SUPABASE_ANON_KEY' with your key

5️⃣  TEST IT OUT
    → Open login.html in browser
    → Or use: python -m http.server 8000
    → Visit: http://localhost:8000/login.html

🆘 TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "Invalid API Key"
   → Check supabase-config.js has correct credentials
   → Remove any extra spaces from keys

❌ CORS Error
   → Use a local server (python -m http.server)
   → Don't open files directly (file://)

❌ Emails Not Sending  
   → Check spam folder
   → Enable email confirmation in Supabase
   → Set up custom SMTP (optional)

❌ OAuth Not Working
   → Enable providers in Supabase dashboard
   → Add correct redirect URLs
   → Configure OAuth apps (Google/GitHub)
