STEP 1: Configure Supabase
─────────────────────────────────────────────────────────────────────

1. Go to https://supabase.com
2. Create a new project (takes ~2 minutes)
3. Go to Settings → API
4. Copy:
   - Project URL
   - Anon Public Key
5. Edit supabase-config.js:

   const SUPABASE_CONFIG = {
       url: 'YOUR_PROJECT_URL_HERE',
       anonKey: 'YOUR_ANON_KEY_HERE',
   };


STEP 2: Start Local Server
─────────────────────────────────────────────────────────────────────

   # Choose one:
   python -m http.server 8000
   # OR
   npx http-server
   # OR
   php -S localhost:8000


STEP 3: Test It!
─────────────────────────────────────────────────────────────────────

1. Open: http://localhost:8000/landing.html
   ✓ Should see professional landing page

2. Try: http://localhost:8000/index.html
   ✓ Should redirect to landing.html (not logged in)

3. Click "Get Started" → Create account
   ✓ Should redirect to index.html after signup

4. Now try: http://localhost:8000/index.html
   ✓ Should stay on index.html (logged in!)

5. Click "Logout"
   ✓ Should redirect to landing.html
